# Extension Components

Extension components are additional components that can be used in conjunction with the existing component blocks in this library. They provide extra functionality and enhance the capabilities of the components.

To use an extension component, simply import it into your project and include it in your component hierarchy. You can then utilize its features and properties to enhance the functionality of your application.

```
import { ChatBubble } from "neos-blocks";
```

With the availability of extension components, you have the flexibility to build powerful and versatile applications using the component blocks provided by this library.
